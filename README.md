# Projeto de Integração: APIs de Software para Metabase (AWS)

Este projeto automatiza a extração de dados de APIs e os centraliza em um banco de dados AWS RDS para visualização no Metabase.

## 🚀 Tecnologias Utilizadas
* **Python**: Extração e tratamento de dados.
* **AWS RDS (PostgreSQL)**: Armazenamento em nuvem.
* **Metabase (EC2)**: Dashboard e BI.
* **GitHub**: Versionamento e documentação.

## 🛠️ Como rodar o projeto
1. Configure sua instância RDS na AWS.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Execute o script: `python main.py`.
4. Conecte o Metabase ao endpoint do seu RDS.
